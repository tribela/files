Simple static file hosting for curl

To upload file:

curl -F file=@hello.txt {{ url_for('upload_file', fname='hello.txt', _external=True) }}

To download file:

curl -O hello.txt {{ url_for('get_file', fname='hello.txt', _external=True) }}

To delete file:

curl -X DELETE {{ url_for('get_file', fname='hello.txt', _external=True) }}
